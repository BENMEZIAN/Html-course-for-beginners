HTML Video

To show a video in HTML, use the <video> element

Example

<video width="320" height="240" controls>
   <source src="audios/Kader Japonais - Bla bik.mp4" type="video/mpeg">
</video>

How it Works

The controls attribute adds video controls, like play, pause, and volume.
It is a good idea to always include width and height attributes. If height and width are not set, the page might flicker while the video loads.
The <source> element allows you to specify alternative video files which the browser may choose from. The browser will use the first recognized format.
The text between the <video> and </video> tags will only be displayed in browsers that do not support the <video> element.

HTML <video> Autoplay

To start a video automatically, use the autoplay attribute

Add muted after autoplay to let your video file start playing automatically (but muted)

<video controls autoplay muted>
   <source src="audios/Kader Japonais - Bla bik.mp4" type="video/mp4">
</video>

Conclusion

Tag        Description

<audio> : Defines a video or movie

<source> : Defines multiple media resources for media elements, such as <video> and <audio>

<track> : Defines text tracks in media players (les traces de texte)
